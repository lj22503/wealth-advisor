## 📋 一、Excel 模板说明

本系统提供 **3 个 Excel 模板**，用于数据导入导出：

| 模板 | 用途 | 必填字段 |
|------|------|----------|
| `客户数据模板.xlsx` | 导入客户基本信息 | 姓名、风险等级 |
| `持仓数据模板.xlsx` | 导入客户持仓 | 客户 ID、基金代码、份额、买入价 |
| `基金数据模板.xlsx` | 导入基金基础数据 | 基金代码、名称、类型、净值 |

## 📝 三、填写指南

### 3.1 客户数据模板

**文件**: `客户数据模板.xlsx`

| 列名 | 说明 | 是否必填 | 示例 | 格式要求 |
|------|------|----------|------|----------|
| A 列 - 客户姓名 | 客户全名 | ✅ 必填 | 张三 | 文本 |
| B 列 - 手机号 | 联系电话 | 可选 | 13800138000 | 11 位数字 |
| C 列 - 邮箱 | 电子邮箱 | 可选 | zhangsan@example.com | 标准邮箱格式 |
| D 列 - 风险等级 | 风险承受能力 | ✅ 必填 | C3 | C1/C2/C3/C4/C5 |
| E 列 - 总资产 | 客户总资产（元） | 可选 | 500000 | 数字，不含逗号 |

**风险等级说明**：
- C1 = 保守型（CONSERVATIVE）
- C2 = 稳健型（MODERATE）
- C3 = 平衡型（MODERATE）
- C4 = 成长型（AGGRESSIVE）
- C5 = 进取型（AGGRESSIVE）

**示例数据**：
```
客户姓名    手机号          邮箱                      风险等级  总资产
张三       13800138000    zhangsan@example.com      C3       500000
李四       13900139000    lisi@example.com          C2       300000
王五       13700137000                              C4       800000
```

### 3.2 持仓数据模板

**文件**: `持仓数据模板.xlsx`

| 列名 | 说明 | 是否必填 | 示例 | 格式要求 |
|------|------|----------|------|----------|
| A 列 - 客户 ID | 客户唯一标识 | ✅ 必填 | client_001 | 文本（与导入客户时的 ID 一致） |
| B 列 - 基金代码 | 基金 6 位代码 | ✅ 必填 | 000001 | 6 位数字 |
| C 列 - 基金名称 | 基金全称 | 可选 | 华夏成长混合 | 文本 |
| D 列 - 持有份额 | 持有数量（份） | ✅ 必填 | 10000 | 数字 |
| E 列 - 买入价格 | 买入时单价（元） | ✅ 必填 | 1.5000 | 数字，保留 4 位小数 |
| F 列 - 买入日期 | 购买日期 | 可选 | 2024-01-15 | YYYY-MM-DD |

**示例数据**：
```
客户 ID      基金代码  基金名称          持有份额   买入价格   买入日期
client_001  000001   华夏成长混合      10000     1.5000    2024-01-15
client_001  000002   易方达蓝筹精选    5000      2.3000    2024-02-20
client_002  000003   招商中证白酒      8000      1.8000    2024-03-10
```

**注意事项**：
1. 客户 ID 必须与「客户数据模板」中的 ID 一致
2. 基金代码必须是 6 位数字
3. 买入价格建议保留 4 位小数（基金净值精度）

## 📤 四、导入流程

### 4.1 导入客户数据

1. 打开飞书应用 → 点击「客户管理」
2. 点击右上角「导入」按钮
3. 选择「客户数据模板.xlsx」
4. 系统自动校验数据
5. 确认导入 → 完成

**校验规则**：
- ✅ 姓名不能为空
- ✅ 风险等级必须是 C1-C5
- ✅ 总资产必须是正数
- ⚠️ 手机号格式不正确会提示警告（但不阻止导入）

### 4.2 导入持仓数据

1. 打开飞书应用 → 点击「持仓管理」
2. 点击右上角「导入」按钮
3. 选择「持仓数据模板.xlsx」
4. 系统自动校验数据
5. 确认导入 → 完成

**校验规则**：
- ✅ 客户 ID 必须已存在
- ✅ 基金代码必须是 6 位数字
- ✅ 持有份额必须 > 0
- ✅ 买入价格必须 > 0

### 4.3 导入基金数据

1. 打开飞书应用 → 点击「基金库」
2. 点击右上角「导入」按钮
3. 选择「基金数据模板.xlsx」
4. 系统自动校验数据
5. 确认导入 → 完成

## 📊 五、导出数据

### 5.1 导出客户列表

1. 进入「客户管理」页面
2. 点击右上角「导出」按钮
3. 选择导出范围（全部/筛选后）
4. 下载 Excel 文件

### 5.2 导出持仓明细

1. 进入「持仓管理」页面
2. 选择客户（可多选）
3. 点击右上角「导出」按钮
4. 下载 Excel 文件

### 5.3 导出诊断报告

1. 进入「持仓诊断」页面
2. 选择客户
3. 点击「生成报告」
4. 点击「导出 Excel」

## 🛠️ 六、常见问题

### Q1: 导入时提示「客户 ID 不存在」
**原因**: 持仓数据中的客户 ID 在系统中没有对应记录  
**解决**: 先导入客户数据，再导入持仓数据

### Q2: 导入时提示「基金代码格式错误」
**原因**: 基金代码不是 6 位数字  
**解决**: 检查基金代码，确保是 6 位数字（如 000001）

### Q3: 风险等级填写什么？
**答案**: 
- 客户风险等级：C1（保守）/ C2（稳健）/ C3（平衡）/ C4（成长）/ C5（进取）
- 基金风险等级：同样使用 C1-C5

### Q4: 可以同时导入多个客户吗？
**答案**: 可以！Excel 模板支持批量导入，每行一个客户/持仓记录

### Q5: 导入失败后数据会丢失吗？
**答案**: 不会。导入采用「全有或全无」策略，校验失败不会写入任何数据

### Q6: 如何更新已有客户数据？
**答案**: 
1. 先导出当前数据
2. 在导出的 Excel 中修改
3. 重新导入（系统会自动更新已有记录）

## 📋 附录：Excel 公式示例

### 计算持仓市值
```excel
= 持有份额 * 当前净值
```

### 计算持仓收益率
```excel
= (当前净值 - 买入价格) / 买入价格 * 100
```

### 计算持仓盈亏
```excel
= (当前净值 - 买入价格) * 持有份额
```

### 计算资产配置比例
```excel
= 某类基金市值 / 总持仓市值 * 100
```
