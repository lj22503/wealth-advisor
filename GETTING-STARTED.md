## 🛠️ 如何让它"能用"起来？

### 方案一：**立即体验原型**（最简单）
```bash
# 直接打开HTML文件
open prototype/F001-home.html
```
✅ **立即看到完整界面**，无需安装任何环境

### 方案二：**运行后端服务**（需要Node.js）
```bash
# 1. 进入后端目录
cd backend

# 2. 安装依赖
npm install

# 3. 启动服务
npm run dev
```
服务运行在：http://localhost:3000

### 方案三：**基于此开发飞书小程序**

**前端开发步骤**：

2. 下载飞书小程序开发工具
3. 基于 `prototype/` 中的设计开发页面
4. 连接本地后端API

## 📊 后端API接口

启动服务后，可以访问以下接口：

1. **健康检查**：`GET /health`
2. **客户列表**：`GET /api/clients`
3. **客户详情**：`GET /api/clients/:id`
4. **创建客户**：`POST /api/clients`
5. **持仓诊断**：`GET /api/clients/:id/diagnosis`
