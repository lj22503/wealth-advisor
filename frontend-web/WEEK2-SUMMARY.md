## 技术亮点

### 1. ECharts 集成
```typescript
// 封装通用图表组件
<Chart option={allocationOption} height={300} />
```

### 2. 表单验证
```typescript
// 基金代码 6 位数字验证
{ pattern: /^\d{6}$/, message: '基金代码为 6 位数字' }

// 手机号格式验证
{ pattern: /^1[3-9]\d{9}$/, message: '请输入正确的手机号' }
```

### 3. 数据格式化
```typescript
// 总资产格式化
formatter={(value) => `¥ ${value}`.replace(/\B(?=(\d{3})+(?!\d))/g, ',')}

// 收益率颜色
color: rate >= 0 ? '#00b42a' : '#f53f3f'
```

### 4. 风险评估算法
```typescript
// 基于分散度和集中度计算风险评分
const diversificationScore = Math.min(100, (fundCount / totalHoldings) * 100);
const concentrationPenalty = maxConcentration > 50 ? 20 : maxConcentration > 30 ? 10 : 0;
const riskScore = Math.max(0, diversificationScore - concentrationPenalty);
```
