# 理财经理投顾系统 PRD V1.0

## 1. 产品概述

### 1.1 产品背景
随着财富管理行业的发展，理财经理需要更专业的工具来服务客户。传统的手工分析效率低下，难以满足客户个性化需求。

### 1.2 产品定位
为理财经理提供一站式投顾工具，提升工作效率和客户服务质量。

### 1.3 目标用户
- 银行理财经理
- 券商投资顾问
- 独立财富机构顾问

## 2. 核心功能

### 2.1 客户管理
- 客户列表展示
- 客户基本信息管理
- 客户资产总览

### 2.2 持仓诊断
- 持仓结构分析
- 风险等级评估
- 资产配置建议

### 2.3 业绩归因
- 收益来源分析
- 风险调整后收益
- 与基准对比

### 2.4 报告生成
- 一键生成客户报告
- 报告模板自定义
- PDF导出功能

## 3. 技术架构

### 3.1 前端架构
- 飞书小程序框架
- 组件化开发
- 响应式设计

### 3.2 后端架构
- 微服务架构
- RESTful API
- 本地数据存储（IndexedDB）
