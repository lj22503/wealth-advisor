## 投资策略建议

### 1. 定投策略
- **金额**：每月定投1万元
- **时间**：每月10日
- **产品**：核心配置基金组合
- **期限**：持续3-5年

### 2. 再平衡策略
- **频率**：每季度再平衡一次
- **阈值**：单一资产偏离目标配置±5%时调整
- **方法**：卖出超配资产，买入低配资产

### 3. 风险控制
- **止损线**：单一基金亏损20%时评估
- **仓位控制**：单只基金不超过总资产的10%
- **行业分散**：单一行业不超过总资产的20%
- **定期回顾**：每季度全面回顾投资组合

### 4. 市场应对
- **牛市**：保持定投，适度止盈
- **熊市**：坚持定投，逢低加仓
- **震荡市**：保持配置，关注结构性机会
- **极端市场**：降低仓位，增加防御性资产
